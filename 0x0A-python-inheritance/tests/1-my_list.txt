==============================
1-my_list
==============================

Print a list in ascending order

==============================

::
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList([1, 5, 2, 3, 4])
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Largest number at the start

::
	>>> my_list = MyList([5, 3, 2, 1, 4])
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

==============================

__str__ method

::
        >>> my_list = MyList([4, 3, 2])
        >>> print(my_list)
        [4, 3, 2]

append() method

::
        >>> my_list = MyList([4, 2, 8])
        >>> my_list.append(5)
        >>> print(my_list)
        [4, 2, 8, 5]

print_sorted() with negative number and unsorted append

::
        >>> my_list = MyList([3, -1, 0])
        >>> my_list.append(2)
        >>> my_list.print_sorted()
        [-1, 0, 2, 3]

print_sorted() with an empty list

::
	>>> my_list = MyList([])
    	>>> my_list.print_sorted()
    	[]

Return a new sorted list without modifying the original

::
        >>> my_list = MyList([10, 5, 3])
        >>> sorted_list = my_list.print_sorted()
        >>> print(sorted_list)
        [3, 5, 10]
        >>> print(my_list)
        [10, 5, 3]
