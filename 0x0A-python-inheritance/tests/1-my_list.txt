Print a list in ascending order

==============================

::
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList([1, 5, 2, 3, 4])
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Largest number at the start

::
	>>> my_list = MyList([5, 3, 2, 1, 4])
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

==============================

### Additional Tests

#### 1. Check __str__ method

::
        >>> my_list = MyList([4, 3, 2])
        >>> print(my_list)
        [4, 3, 2]  # Ensure the __str__ method outputs the original list

#### 2. Check append() method

::
        >>> my_list = MyList([4, 2, 8])
        >>> my_list.append(5)
        >>> print(my_list)
        [4, 2, 8, 5]  # Ensure the new item is appended correctly

#### 3. Check print_sorted() with negative number and unsorted append

::
        >>> my_list = MyList([3, -1, 0])
        >>> my_list.append(2)
        >>> my_list.print_sorted()
        [-1, 0, 2, 3]  # Ensure negative and positive numbers are sorted

#### 4. Check print_sorted() with an empty list

::
        >>> my_list = MyList([])
        >>> my_list.print_sorted()
        []  # Ensure nothing breaks when the list is empty

#### 5. Check print_sorted() returns a new sorted list without modifying the original

::
        >>> my_list = MyList([10, 5, 3])
        >>> sorted_list = my_list.print_sorted()
        >>> print(sorted_list)
        [3, 5, 10]  # Ensure print_sorted returns a new sorted list
        >>> print(my_list)
        [10, 5, 3]  # Ensure original list is unmodified
